# 🏥 HealthClaim Analytics — Power BI Dashboard

> A 6-page interactive Power BI dashboard analysing 52,000+ synthetic 
> patient claims across 20 Nigerian healthcare providers (2020–2024).

---


## 📊 Dashboard Pages

| Page | Purpose |
|---|---|
| Executive Summary | KPIs, claim trends, provider overview |
| Claims Performance | Cost drivers by service type and diagnosis |
| Fraud & Anomaly Detection | Outlier claims, duplicates, approval anomalies |
| Operational Efficiency | SLA tracking, processing bottlenecks |
| Key Insights | Data-driven findings summary |
| Recommendations | Actionable next steps from the analysis |

---

## 🔍 Key Findings

- 💰 Surgery averages ₦339,262 per claim — 16x more than Consultation
- 🚨 1,400 high-value claims flagged as anomalies
- 📋 8 duplicate claim patterns detected (same patient, date, service)
- ⏱ Only 14% of claims met the 14-day SLA target
- 📌 Pending claims took 27 days avg — 4x slower than approved claims
- 🏥 FMC Owerri led total spend | FMC Calabar led patient volume

## 🛠 Tools & Skills Used

| Tool | Usage |
|---|---|

| DAX | KPI measures, SLA rate, duplicate detection |
| Power Query (M) | Data cleaning, bucket columns, transformations |

## 📁 Dataset Structure

| Table | Rows | Description |
|---|---|---|
| Claims Data | 52,000 | Core claims with amount, type, status, provider |
| Provider Data | 20 | Hospital type, location, patient volume |
| Time Dimension | 1,827 | Date table for time intelligence |
